#        Gradebook Analyzer (Python)

A simple Python command-line tool for managing and analyzing student grades. This project demonstrates fundamental Python skills, including file I/O, data structures (dictionaries and lists), modular functions, list comprehensions, and formatted output.

-------------------
Features
-------------------

* Flexible Data Input: Enter student names and marks manually via the CLI.
* CSV Import: Load data directly from a .csv file.
* Statistical Analysis: Automatically calculates:
    * Class Average
    * Median Score
    * Highest Score (and the student who achieved it)
    * Lowest Score (and the student who achieved it)
* Grade Assignment: Assigns letter grades (A-F) based on a standard scale.
* Pass/Fail Filtering: Quickly see lists of students who passed or failed based on a defined threshold (default: 40).
* Full Report: Displays a clean, formatted table of all students, their marks, and their final grades.
* CSV Export: Save the final formatted report to a new CSV file.

-------------------

No external libraries are needed. The script uses only standard Python modules
